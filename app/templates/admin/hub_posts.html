Como usar nos 3 sites

Fluxo ideal para você duplicar este projeto sem manter 3 painéis diferentes.

  1. Escolha um projeto para ser o painel principal.
  2. Nos outros projetos duplicados, entre em Hub Posts, ative o recurso, defina um site key e um token para receber.
  3. No painel principal, cadastre a URL e o token de cada espelho acima.
  4. Faça as matérias somente no painel principal. Ao salvar, o sistema já envia para os espelhos.
  5. Depois você só troca logo, domínio e visual da home de cada espelho.
